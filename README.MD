## Development

This repository is a pnpm monorepo managed with Turborepo.

```bash
# Clone the repository
git clone https://github.com/Assylum-Labs/hermis.git

# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run tests
pnpm test
```
